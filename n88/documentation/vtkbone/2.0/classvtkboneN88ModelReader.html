Detailed Description

read n88model files

vtkboneN88ModelReader is a source object that reads .n88model files that represent finite element models.

vtkboneN88ModelReader creates a vtkboneFiniteElementModel dataset.

See also
vtkboneFiniteElementModel vtkboneN88ModelWriter
