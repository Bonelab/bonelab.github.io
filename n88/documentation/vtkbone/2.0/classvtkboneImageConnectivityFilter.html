Detailed Description

mask regions of image data based on geometric connectivity

This filter uses vtkboneImageConnectivityMap to generate a mask based on connectivity of the scalar values of the image. Specified components can then be zeroed out of the image.

This filter accepts image scalars on either the cells or the points; the output will have scalars with the same association.
