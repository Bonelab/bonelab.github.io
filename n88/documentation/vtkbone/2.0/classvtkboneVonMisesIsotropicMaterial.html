Detailed Description

An object representing an isotropic Mohr Coulomb elastoplastic material.

An object representing an isotropic von Mises elastoplastic material.

vtkboneMohrCoulombIsotropicMaterial stores the parameters for an isotropic material with Mohr Coulomb elastoplastic properties.

See also
vtkboneMaterialTable vtkboneFiniteElementModel vtkboneGenerateHomogeneousMohrCoulombIsotropicMaterialTable vtkboneGenerateHommingaMaterialTable vtkboneLinearOrthotropicMaterial vtkboneLinearAnistropicMaterial vtkboneVonMisesIsotropicMaterial

vtkboneVonMisesIsotropicMaterial stores the parameters for an isotropic material with von Mises elastoplastic properties.
