Detailed Description

An abstract base class for materials of materials of the same type.

Note that although this represents a collection of similar materials, MaterialArray is not implemented an array of material objects, but as a single object that inherits from vtkboneMaterial, and that stores arrays of material properties. This more closely matches how material arrays are stored in n88model files, and also how they are employed by the solver.
